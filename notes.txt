Create a virtual environment
run: python -m venv venv
this will create a folder named: venv/

Activate the environment
venv\Scripts\activate
you will see something like :(venv) C:\Users\USER\Web development\Meme Hand Tracking>

Install reqirements.txt
run: pip install -r requirements.txt
datasets: https://www.kaggle.com/datasets/lynnangelaclawagon/meme-hand-tracker/data

Run the application:
python app.py
